# ЛК студента РУК

Frontend личного кабинета Российского университета кооперации. Пока без backend — всё на моках, но структура рассчитана на подключение API.

## Запуск

```bash
npm install
npm run dev
```

Продакшен-сборка: `npm run build`

**Вход для проверки:** почта с `@`, пароль от 4 символов, код — любые 6 цифр.

## Что где лежит

- `src/pages/` — экраны
- `src/mocks/` — данные для разработки
- `src/ui/` — кнопки, таблицы, модалки
- `src/layout/` — вход, кабинет, меню
- `auth.ts`, `study.ts`, `settings.ts` — сессия, программа обучения, настройки
- `apiClient.ts` — запросы к API (когда появится backend)
- `styles/tokens.css` — цвета и отступы

Импорты через `@/` → `src/`.

## Backend

Скопируй `.env.example` в `.env` и пропиши адрес API:

```
VITE_API_BASE_URL=http://localhost:8080
```

Дальше — заменять моки на вызовы из `apiClient.ts` и перевести авторизацию с mock на SSO/cookie. Пока переменная пустая, приложение живёт только на моках.

**Деплой на сервер:** Docker Compose — [`deploy/DEPLOY.md`](deploy/DEPLOY.md).
